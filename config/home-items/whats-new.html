We will be updating our reports with new functionality and content regularly. Information on what’s changed will be published here.

20/02/2018
  • A new report Scoreboard has been added to the advanced promotion collection. This report combines financial and customer metrics to bring together all the information about an individual promotion in one place.
  • Exporting to Excel functionality has been added to all reports!
10/02/2018
  • Additional functionality has been added in the advanced promotion module which allows you to select up to five categories. This functionality is across all reports
  • Within the Databank report you can now select either a category or a mechanic to find your results
01/02/2018
  • Data has been refreshed across all reports to include promotions with an end date up to the 30th of January, 2018
  • The Databank report includes functionality to filter the promotion column by any string (e.g. Prego). You can also keep the selection from one search (e.g. 2/$3) and add the selection from searching for a different string (e.g. 3/$5)
21/01/2018
  • The Landscape report has new functionality. Clicking the ‘Line Chart’ button will display a line chart showing the total sales and % of sales on deal corresponding to the gantt chart. You can switch the measures displayed in the line chart to focus on promotion only sales if desired.
  • Clicking on a promotion in the Landscape report will display a button labelled ‘Products and Stores’, clicking this button will provide a pop-up detailing the products and stores associated with the selected promotion
  • Duration in the Databank report is now measured in days
  • In the Category Contribution report when you select a measure to display the column chart of promotions the promotion name will include the start date
08/01/2018
  • Sorting in tables will now order from highest to lowest on the first click
  • In the Category Contribution report all positive uplift measures will be green and negative uplift measures red in the waterfall chart
  • Unwanted segmentations in the Customer Profile report have been removed
  • In the Landscape the colours of the legend items will be in the same order
  • Increased Repertoire and Switchers measures in Loyalty Dynamics have been renamed to Increased Category Purchases and Decreased Category Purchases
  • In the Loyalty Dynamics report the colours are now blue and orange instead of red and green
